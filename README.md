
# PointBlank Server V41

Server PointBlank Turkey V41 Based on Node.js (ON WIP)


## Installation

Install dependency used with npm

```bash
  npm install
```
    
Build the project with TypeScript

```bash
  tsc
```

Run the server

```bash
  node ./dist/server.js
```
